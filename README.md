# Job Portal Project 💼

This is a simple Job Portal website built using HTML, CSS, and JavaScript.

## 🚀 Features
- Job listings display
- Search functionality
- Apply button with popup form
- Dark and Light mode toggle
- Responsive design (basic)

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📂 Project Structure
- index.html
- style.css
- script.js

## 🎯 Future Improvements
- Save applied jobs
- Backend integration
- User login system
